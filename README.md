# OrbitFriends Capstone Project

OrbitFriends is a hobby friend finder with a dating-app-style browsing flow.

## Features

- account creation and login
- profile setup with username, hobbies, and bio
- browse community profiles with Interested and Not Interested actions
- match creation based on shared hobbies
- one-on-one chat for matches
- group creation using existing matches
- local storage persistence so data stays after refresh
